# 32-bit RISC-V CPU (RV32I)

This project implements a basic 32-bit RISC-V processor using SystemVerilog.

Features:
- RV32I instruction subset
- Single cycle CPU
- 32 register register-file
- ALU with arithmetic and logic operations
- Instruction and data memory
- Testbench for simulation

Tools:
- ModelSim (Simulation)
- Yosys (Synthesis)
- OpenROAD (Physical Design)
